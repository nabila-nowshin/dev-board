Question-1: What are the different ways to select an element in the DOM?

  • document.getElementById("id")
  • document.getElementsByClassName("class")
  • document.getElementsByTagName("tag")
  • document.querySelector("selector")
  • document.querySelectorAll("selector")

Question-2: What is the difference between innerHTML, innerText, and textContent ?

  • innerHTML - Gets or sets HTML & text(includes tags).
  • innerText - Gets or sets visible text only.
  • textContent - Gets or sets all text (including hidden content).

Question-3: What is event delegation in the DOM?

Event delegation is a technique where single event listener is attached to a parent element instead of adding listeners to multiple child elements. The event bubbles up from the child, allowing the parent to handle it.

Question-4: What is event bubbling in the DOM?

Event bubbling is a mechanism that allows events to propagate up the Document Object Model (DOM) tree. It's a fundamental concept in event propagation.

Question-5: How do you create, add, and remove elements using JavaScript?

  • Create an element : document.createElement("tag")
  • Adds an element : parent.appendChild(element)
  • Removes an element:element.remove()
